# 复盘输出模板

## 标准模板（通用）

```
已复盘项目上下文：

- **项目**：{项目名}（{技术栈}）
- **版本**：v{x.y.z}（git: {hash}）
- **最近**：{最近 1-3 个变更}
- **坑点**：{关键坑点 1-2 个}
- **待办**：{Next 计划，如无则省略}

有什么可以帮你的？
```

## 示例：萤石云项目

```
已复盘项目上下文：

- **项目**：萤石云视频监控 H5 客户端（Next.js + React + TS）
- **版本**：v0.2.4（git: 4942c37）
- **最近**：API 错误处理优化、全屏控制统一化、移动端播放器优化
- **坑点**：playerKey 不要加时间戳（会导致 resize 循环）
- **待办**：调研回放功能开通步骤

有什么可以帮你的？
```

## 示例：无 Daily Notes 的项目

```
已了解项目：

- **项目**：xxx（技术栈从 package.json）
- **说明**：从 README.md 的一句话描述
- **结构**：关键目录说明

今天有什么我可以帮你的？
```

## 示例：极简版（用户着急时）

```
已复盘。当前 v0.2.4，最近优化了 API 错误处理和播放器。
有什么可以帮你的？
```
