## Naming Convention

All items use **timestamp IDs** to avoid collision without a central database:

| Type | Example |
|------|---------|
| ADR  | `ADR-20260423101938000-thin-skill-pattern.md` |
| Epic | `EPIC-20260423102000000-lythoskill-bootstrap.md` |
| Task | `TASK-20260423124059736-create-templates.md` |

Format: `PREFIX-yyyyMMddHHmmssSSS` (17 digits).

## How to Create Items

Use the CLI (do not create files by hand):

```bash
bunx @lythos/project-cortex adr  "Your decision title"
bunx @lythos/project-cortex epic "Your epic title"
bunx @lythos/project-cortex task "Your task title"
bunx @lythos/project-cortex wiki "Your knowledge entry title" --category pattern|faq|lesson
```

The CLI auto-assigns timestamp IDs, generates templates, and places files in the correct status directory.
