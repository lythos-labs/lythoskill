# @lythos/skill-coach

> Skill quality reviewer. Analyzes SKILL.md files against Agent Skills best practices and provides actionable optimization advice.

## Why

Writing a good SKILL.md is harder than it looks. This skill codifies the evaluation criteria — body size, description quality, progressive disclosure, reference hygiene, frontmatter hygiene, one-skill-one-job, factual accuracy, and the naive-agent test — into a repeatable review process.

## Skill Documentation

This package is primarily a **Skill** layer — the agent-visible documentation is here:  
[packages/lythoskill-skill-coach/skill/SKILL.md](../../packages/lythoskill-skill-coach/skill/SKILL.md)

There is no CLI starter layer; the value is entirely in the SKILL.md and its reference materials.
