# @lythos/skill-hello-world

> The thinnest possible lythoskill skill. Zero scripts, just a SKILL.md.

Part of the [lythoskill](https://github.com/lythos-labs/lythoskill) meta-skill ecosystem.

## What it does

Demonstrates the minimum viable skill: a single `SKILL.md` with YAML frontmatter, no dependencies, no scripts. Useful as a template when starting a new skill from scratch.

## Architecture

```
lythoskill-hello-world/
└── SKILL.md   # Intent + usage instructions only
```

This is the **Skill** layer with no **Starter** — the simplest form of the thin-skill pattern.
