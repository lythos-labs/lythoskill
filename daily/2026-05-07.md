## 4. 关键认知：skill 路径的三种模式

| 模式 | 示例 | 说明 |
|------|------|------|
| Repo root | `github.com/garrytan/gstack` | SKILL.md 直接在 repo root |
| Monorepo | `github.com/lythos-labs/lythoskill/skills/lythoskill-deck` | repo 内部有 `skills/` 目录 |
| Double skills | `github.com/mattpocock/skills/skills/engineering/tdd` | repo 名也叫 skills，内部也有 skills/ |

**Rule**: `path` = repo root 到 SKILL.md 所在目录的精确相对路径。不省略、不猜测、不自动补全。
